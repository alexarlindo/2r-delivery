# Rápido Rápido • Mockup para Motoboys

Mockup estático de um painel mobile para motoboys da Rápido Rápido.

O protótipo simula um fluxo simples de entregas:

- visão inicial com resumo do dia;
- lista de entregas por fazer;
- criação de nova entrega;
- assinatura do cliente no canvas;
- envio automático da entrega para a lista de concluídas;
- perfil básico do entregador.

## Tecnologias

- HTML
- CSS
- JavaScript
- Tabler Icons via CDN

## Como executar

Abra o arquivo `index.html` no navegador.

Também pode usar qualquer servidor estático simples, por exemplo:

```bash
python3 -m http.server 3000
```

Depois acesse `http://localhost:3000`.

## Escopo

Este projeto é apenas um mockup visual e funcional. Ele não possui backend, autenticação, banco de dados ou persistência real das entregas.
